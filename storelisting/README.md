# Store Listing Metadata
All files in this directory contain copy for App Store and Google Play listings.

## Files

- `ios-subtitle.txt` — App Store subtitle (30 characters max)
- `ios-description.txt` — Full App Store description (~4000 characters)
- `ios-keywords.txt` — Comma-separated keywords (100 characters max)
- `android-short-description.txt` — Google Play short description (80 characters)
- `android-full-description.txt` — Google Play full description (~4000 characters)
- `android-keywords.txt` — Google Play keyword list (separate field, 30 chars each)
- `release-notes-v1.0.0.txt` — "What's New" for version 1.0.0

## Notes

- All content subject to review and copy-editing before submission
- Screenshot text overlays should mirror key phrases from these descriptions
- Keywords should match App Store optimization (ASO) strategy
- Release notes should be concise (max 500 characters recommended)
